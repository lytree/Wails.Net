测试方法

1. 在终端中再次运行 dotnet run --project examples/Wails.Net.Demo.SingleInstance/...，或直接运行已编译的可执行文件。

2. 第二个进程会立即退出，首个实例的窗口被聚焦，历史新增一条 second 记录。

3. 可附加命令行参数（如 --foo bar），参数会通过 OnSecondInstanceLaunch 回调传给首实例并显示在历史中。

说明

配置：SingleInstance=true + SingleInstanceUniqueID（可选，默认回退到应用名）。

平台层通过 IPlatformApp.AcquireSingleInstanceLock(uniqueId) 获取锁，失败时调用 NotifySingleInstance(args) 通知首实例后退出。

首实例通过 Application.OnSecondInstanceLaunch(Action<string[]>) 注册回调，框架同时广播 wails:second-instance:launched 事件。
