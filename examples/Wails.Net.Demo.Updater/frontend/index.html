说明

多 Provider 模式：在 UpdaterService 中注册多个 IUpdateProvider，按注册顺序依次尝试，首个返回非 null 清单者胜出。

事件：wails:updater:update-available / no-update / download-started / download-progress / download-complete / install-started / install-complete。

实际应用中应使用 HttpUpdateProvider / GitHubUpdateProvider / GitLabUpdateProvider，并配置 UpdaterConfig.UpdateURL。
